# 🔭 Needle-Moving Work

Needle-moving work refers to identifying and executing on longer-term initiatives that might not have immediate impact, but create magnitude shifts in product outcomes when they do.

**Underlying issue:** PMs are stuck in incrementalism — spending the majority of sprints implementing small, quickly-implementable changes that yield only minor improvements.
