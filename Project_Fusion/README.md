# 🌀 Project Fusion — Product Coaching

A product coaching framework built around the six core leverages that separate great product managers from good ones.

---

## Six Leverages of a Product Manager

| # | Leverage | Core Idea |
|---|----------|-----------|
| 1 | [Leadership Buy-in](./01-leadership-buy-in.md) | Secure time, resources, and executive support |
| 2 | [Team Empowerment](./02-team-empowerment.md) | Enable the team to make great decisions without you |
| 3 | [Needle-Moving Work](./03-needle-moving-work/) | Identify and execute initiatives that create magnitude shifts |
| 4 | [Focused Work](./04-focused-work/) | Concentrate effort on what has the highest leverage |
| 5 | [Product Intuition](./05-product-intuition/) | Develop instinct through deep customer exposure |
| 6 | [PM Leadership](./06-pm-leadership/) | Lead the team and set goals that drive commercial value |
